PS2Dox Reference Manual - Operations

Operations

Operations in PS2Dox encompass various actions and processes within the system.

Categories of operations:

  1. User Management
    • User registration
    • Authentication
    • Profile management
  2. Portfolio Management
    • Portfolio creation
    • Portfolio updates
    • Portfolio deletion
  3. Trading
    • Order placement
    • Order modification
    • Order cancellation
  4. Data Retrieval
    • Market data fetching
    • Performance metrics calculation
    • Report generation
  5. System Administration
    • User account management
    • System configuration
    • Performance monitoring

Help Text: Operations are the core functionalities of PS2Dox. They allow you to manage users, portfolios, trades, and system configurations. Each operation category has its own set of commands and permissions.
