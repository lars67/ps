PS2Dox Reference Manual - Console

Console

The PS2Dox console provides a command-line interface for interacting with the system.

Features:

  • Command execution
  • Output display
  • Error reporting
  • System status monitoring

Help Text: The console is your primary interface for interacting with PS2Dox. It allows you to execute commands, view system outputs, and monitor the status of various operations. Proper authentication is required to access the console.
