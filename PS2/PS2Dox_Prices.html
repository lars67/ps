PS2Dox Reference Manual - Prices

Prices

The Prices module in PS2Dox handles asset pricing data.

Price Operations

1. Get Current Price

Command:

{
  "command": "prices.getCurrent",
  "symbol": "AAPL",
  "exchange": "NASDAQ"
}

Parameters:

Parameter Description Required
symbol Ticker symbol of the asset Yes
exchange Exchange where the asset is traded Yes

Output: Returns an object with the current price and timestamp.

Help Text: Retrieves the most recent price for a specified symbol. Ensure you provide the correct exchange for accurate pricing.

2. Get Historical Prices

Command:

{
  "command": "prices.getHistorical",
  "symbol": "GOOGL",
  "startDate": "2023-01-01",
  "endDate": "2023-12-31",
  "interval": "daily"
}

Parameters:

Parameter Description Required
symbol Ticker symbol of the asset Yes
startDate Start date for historical data (YYYY-MM-DD) Yes
endDate End date for historical data (YYYY-MM-DD) Yes
interval Data interval (daily, weekly, monthly) Yes

Output: Returns an array of objects, each containing a date and corresponding price data.

Help Text: Fetches historical price data for a symbol over a specified date range. The interval can be set to daily, weekly, or monthly.

3. Set Price Alert

Command:

{
  "command": "prices.setAlert",
  "symbol": "TSLA",
  "condition": "above",
  "price": 800.00
}

Parameters:

Parameter Description Required
symbol Ticker symbol of the asset Yes
condition Alert condition (above, below) Yes
price Target price for the alert Yes

Output: Returns a confirmation object with the alert details.

Help Text: Creates a price alert for a specific symbol. You'll be notified when the price meets the specified condition.

4. Manage Price Feed

Command:

{
  "command": "prices.manageFeed",
  "action": "subscribe",
  "symbols": ["AAPL", "GOOGL", "MSFT"]
}

Parameters:

Parameter Description Required
action Action to perform (subscribe, unsubscribe) Yes
symbols Array of ticker symbols Yes

Output: Returns a confirmation object with the updated subscription status.

Help Text: Allows you to subscribe to or unsubscribe from real-time price feeds for specified symbols. This affects the data you receive in real-time updates.
