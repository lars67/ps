Put Cash

Put/Get cash to/from portfolio

Parameter Value Required
portfolioId Portfolio selection, can be used portfolio fields: _id or name or accountId Yes
amount value in currency Yes
currency value . Example USD,DKK,EUR... Yes
rate Cash currency rate. When rate is not set and currency is not portfolio.currency will be used currency rate from historical dat according tradeTime. Rate used for calculate value this currency in base currency No
tradeTime When not set will be used current time. Format: "tradeTime" :"2024-03-05T08:00:00" No
fee Fee for command in current currency. Fee applied to portfolio currency as -fee*rate No
description Description No
tradeType Trade Type: cash|dividends|investment|correction. Default type : cash No
aml AML No
tradeId Some external system id No

Examples:

{"command":"portfolios.putCash","portfolioId":"_testCashPortfolio","amount":"1000","currency":"EUR", "fee":"10", "rate":"1.2"}
Use rate. Will be applied to base currency as - rate*fee
Currency EUR tracked in base curency as +rate*amount
{"command":"portfolios.putCash","portfolioId":"_testCashPortfolio","tradeId":"t3","amount":"-100000","currency":"DKK","rate":"1","tradeType":"correction","description":"put correction"}
Use tradeType = correction. It work similar to other tradeTypes : cash, dividends, investment. only set tradeType.
Used rate=1 and currency DKK ,when portfolio currency is not DKK it is mean that amount in portfolio currency. As far cashes tracked in portfolio currency. Cash in portfolio currency calculated as amount*rate.
{"command":"portfolios.putCash","portfolioId":"_testCashPortfolio","amount":"-1000","currency":"DKK", "tradeTime" :"2024-03-06T08:00:00"}
Used cammand with currency DKK which is not base currency and without rate. In this case rate will be requested from historical data. Returnrd rate is equal command:
{"command":"prices.historical", "symbols": "DKKUSD:FX", "date":"2024-03-06}
After this operation CASH DKK tracked in portfolio currency(USD) will be -1000*0.1456, where DKKUSD:FX=0.1456
